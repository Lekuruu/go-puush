What are pools?

A pool is like an album of content. You can sort your uploads into pools to both organise them and restrict access.

What is an API key and when should I reset it?

An API key is an alternative to your password which is used by the {{ .Config.Service.Name }} client apps. It keeps your password secure while allowing authenticated communications. You can reset your API key if you believe your PC has been compromised or stolen.
